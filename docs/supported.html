Supported Regression models

The package currently support various object scikit-learn <https://scikit-learn.org/stable/> ` and has limited support for `Keras <https://keras.io/> and PyTorch <https://pytorch.org/> neural networks with ReLU activation.

Scikit-learn

The following tables list the name of the models supported, the name of the corresponding object in the python framework, and the function that can be used to insert it in a gurobi model
