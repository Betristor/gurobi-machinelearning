Examples

We develop here examples that showcase specific applications where Gurobi Machine Learning can be used. Each example should be self-contained, and explains different aspects of using the package.

In Surrogate Models, we show how to approximate a nonlinear function with a neural network to use it in a Gurobi model.

In Adversarial Machine Learning, we show how to build a model to construct adversarial examples on the MNIST digit database.

In Student Enrollment, we use a logistic regression and embed it in a Gurobi model.

In Price Optimization, we use a linear regression with categorical features and show how it can be embedded in a Gurobi model 1.

Footnotes

1

Support for categorical feature is very limited and in particular they can’t be used easily as variables.
